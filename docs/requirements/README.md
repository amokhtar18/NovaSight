## Key Architectural Principles

### 🔒 The Template Engine Rule

> **NO Arbitrary Code Generation**: The system MUST NOT generate raw code from scratch using LLM or user input.
>
> **Template-Filling Only**: All executable artifacts are generated by filling pre-approved, security-audited templates with validated user inputs.

See [ADR-002: Template-Filling Architecture](Architecture_Decisions.md#adr-002-template-filling-architecture) for complete details.
